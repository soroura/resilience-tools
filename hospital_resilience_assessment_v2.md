**Based on: Resilient Hospitals - Inter-regional Guidance for Strengthening Resilience to Health Emergencies and Disasters**

---

## Instructions

**Scoring System:**
| Status | Score | Description |
|--------|-------|-------------|
| **Not Started** | 0 | No action taken, not implemented |
| **In Progress** | 0.5 | In progress, partially implemented, or needs improvement |
| **Completed** | 1 | Fully implemented and regularly maintained |
| **N/A** | — | Not applicable (excluded from scoring) |

**Column Definitions:**
| Column | Description |
|--------|-------------|
| **Evidence of Compliance** | Required documentation or proof for each component |
| **Status** | Mark one: Not Started (0), In Progress (0.5), Completed (1), or N/A |
| **Corrective Action** | Required actions to achieve compliance (mandatory if not Completed) |
| **Responsible** | Person/department accountable for this item |
| **Comment** | Additional notes (required if N/A to justify exclusion) |
| **Due Date** | Target completion date |
| **Days Remaining** | Auto-calculated: Due Date - Today |
